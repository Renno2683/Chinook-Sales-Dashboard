## 🎯 Project Overview

An interactive, fully browser-based sales dashboard built for the **Chinook digital music store** dataset. This project visualises key business metrics across revenue, customers, genres, and employee performance — designed to simulate what a real Data Analyst would present to business stakeholders.

## 📈 What the Dashboard Shows

| Section | Description |
|---|---|
| 💰 **KPI Cards** | Total revenue, orders, unique customers, average order value |
| 📈 **Monthly Revenue Trend** | Line chart tracking sales from 2009–2013 |
| 🎸 **Genre Breakdown** | Donut chart of purchases by music genre |
| 🌍 **Revenue by Country** | Horizontal bar chart of top 8 markets |
| 👤 **Top Customers** | Table of highest spenders with visual spend bars |
| 🧑‍💼 **Sales Agent Performance** | Revenue comparison across support agents |

---

## 🔍 Key Insights

- 🇺🇸 **USA is the #1 market** generating $523.06 — nearly double the next country (Canada at $303.96)
- 🎸 **Rock dominates** with 835 purchases, more than double the second-placed genre (Latin, 386)
- 💡 **Top customer Helena Holý** spent $49.62 — the highest of all 59 customers
- 🏆 **Jane Peacock** leads the sales team with $833.04 in revenue generated
- 📦 Average order value is **$5.65**, reflecting a digital download pricing model
